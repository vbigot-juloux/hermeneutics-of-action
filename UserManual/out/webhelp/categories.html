Analytical taxonomies

Note: xPath: /teiCorpus/TEI/teiHeader/encodingDesc
  • xPath XML-TEI elements: teiCorpus|TEI|teiHeader|encodingDesc

Three main taxonomies are created for investigating a narrative text:

  1. Primary data
  2. Objective data
  3. Subjective data
These taxonomies are described within two encoding description sections: 

<encodingDesc><classDecl>
   <taxonomy xml:id="primaryData">  <!-- Primary Data --> </taxonomy> 
   <taxonomy xml:id="objectiveData">  <!-- Objective variables --> </taxonomy> 
   <taxonomy xml:id="subjectiveData">  <!-- Subjective variables --> </taxonomy> 
   </classDecl></encodingDesc>

In anticipation of the ontology (OWL) of power relationships with linked open data standards, I use as XML-TEI attributes few OWL class rdf:ID of the Resource Description Framework (RDF) proposed by OntoMedia ontology. The OWL class is similar to the XML attribute, which is of course a very valuable argument. When a OWL class is used for XML-TEI attribute, it will be specified.

My choice of using the OntoMedia ontology is relevant because we share the same primary goal, althought final purpose is different since mine is for anthropological purpose: "to allow people to explore and analyse narratives" .
